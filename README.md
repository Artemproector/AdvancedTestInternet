# AdvancedTestInternet

**AdvancedTestInternet** — расширенный тест интернета. Позволяет проводить проверку скорости, доступности и режима работы сети. Проект может обнаружить работу сети в режиме белых списков, черных списков, проверить наличие или отсутствие доступа в интернет.

---

## Возможности

- **Проверка скорости** — измерение входящей и исходящей скорости интернета
- **Тест пинга** — измерение задержки до серверов
- **Определение режима сети** — автоматическое определение режима: «Полный доступ», «Белые списки», «Черные списки» или «Нет интернета»
- **История проверок** — сохранение результатов всех тестов

---

## Установка и запуск

1. Перейдите на страницу [Releases](https://github.com/Artemproector/AdvancedTestInternet/releases)
2. Скачайте последнюю версию
3. Разархивируйте скаченный файл
4. Откройте файл `index.html` в любом **браузере** (важно открывать именно в браузере, а не через `Средство просмотра html` или подобный ридер)

---

## Как пользоваться

1. Откройте приложение в **браузере**
2. Нажмите кнопку **«Запустить тест»**
3. Дождитесь завершения проверки
4. Ознакомьтесь с результатами:
   - **Входящая скорость** — скорость скачивания данных
   - **Исходящая скорость** — скорость отправки данных
   - **Пинг** — задержка до сервера
   - **Режим сети** — текущий режим работы интернета
5. Проверьте доступность категорий сайтов в блоке «Доступность категорий»
6. Для просмотра истории проверок откройте меню (кнопка с тремя полосками) и выберите **«История проверок»**

## Технологии

- HTML, CSS, JavaScript (Vanilla JS)
- Fetch API для сетевых запросов
- localStorage для хранения истории проверок
